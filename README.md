おじさん
====
### 使い方

- ### githubからダウンロード

`git clone git://github.com/fastec02/Ojisan_docker`

- ### Docker

#### Dockerのビルド

`docker-compose build`

#### Dockerの起動

`docker-compose up -d`

#### Dockerの作業シェルに入る

`docker exec -it node /bin/sh`

- ### npm

#### npmのインストール

`npm install`

#### npmサーバの起動

`npm run serve`
